Vim commands (Draft)

  • To move left, right, up, and down
  • To delete a letter _ x
  • To undo _ u
  • To move to the end of the line and to put in insert move _ shift+a
  • To delete a word _ d+w
  • To delete the whole line _ d+d
  • To delete the whole line from the cursor postion _ d+shift+$
  • To the beginning of the line _ 0
  • To the end of the line _ shift+$
  • To move to number of words _ no+w (eg:3w will move 3 words from the start)
  • To move to number of words backwords _ no+b (eg:3b will go backwords 3 words)
  • To move to end of the word _ e
  • To move to end of the number of words forward _ 3e
  • To delete number of words _ 3+d+w or d+3+w
  • To undo all the changes in a line _ shift+u
  • To redo _ ctrl+r
  • To move a line around _ dd , p (when it delete a line, it will not completely forget about the line it just deleted. It will put it in some kinda of a registery, so you could move the line where u want to paste the line and type p)
  • To replace a letter _ r on the letter and then type the replacement letter
  • To change the word and to place you in insert mode _ ce
  • To change the whole line and to place u in inser mode _ c$
  • To change certain number of words _ c3w
  • To move to the bottom of the file _ shift+g
  • To get information about line and column number _ ctrl+G
  • To go to specific line with line number _ 4+shift+G (will move the cursor to the 4th line)
  • To go the very start of the file _ gg
  • Search, /whattosearch n to move next, shift+n to move backwards, ctrl+o go back to where u came from, ctrl+I move forwards, ?whatosearch to search bottom up
  • shift+% to move back and forth between paranthesis
  • Substitution :s/old/new/g g for globally in a line, without g will replace only one occurence
  • #,#s/old/new/g to replace things within specific line range
  • %s/old/new/g to change in whole file
  • %s/old/new/gc prompt before changing it
  • To execute external commands :!ls
  • To write to another file :w test.txt
  • You can chaing commands :!rm test.txt after removing the file if you want to list out the file again instead of pressing enter key type in the other command :!ls
  • To select text go to visual mode. When you are in visual mode the path of your cursor will be highlighted so that you could easily delete the selected ones for instance.
  • Selected text could be written to other files by typing :<>w test.txt
  • To reterive and merage files, place the cursor on where you want the text then press :r test.txt, this will place whatever is in test.txt in the cursor position
  • To paste the output from a command to the file :r !ls
  • To create a brand new line o, It will bring u to the new line right after your cursor placement and put you in insert mode
  • To add a line above shift+o
  • a to append text after the cursor position
  • shift+r to go to replace mode
  • copy paste: select the text, press y to yank then p to paste it. To want a word press yw, y2w is to yank 2 words
  • Set option: :set ic (to ignore case), /ignore will also highlight all of them ignorning cases
    • :set hls is(incrementalsearch) hls for higlight
    • to turn off features :set noic
    • set nohls nois
  • To move between text file ctrl+w+w
  • :e test.txt to do autocompletion to get the file that starts with letter t to edit the file
  • f type the character you wanna go to, ; to go to the next match
  • To stop in the character right before t +the character to match
  • 2fa to move the 2 matching character
  • to go to specific line 2000 shift+g
  • ctrl+F for moving forward through the document
  • ctrl+B for moving backward through the document
  • Ctrl+U to move half page backward
  • Ctrl+D to move hald pafe forward
  • Shift+M to move to middle of the screen
  • Shift+H to move to head
  • Shift+L to move to the end of the screen
  • Shift+* to search for the current letter
  • Shift+V line selection
  • visual mode : norm i# to comment block of codes
  • :norm x to remove comments
  • ctrl+v to form visual block selection
  • ctrl +v , shift+i (for insert mode), shift+3, escape for commenting
  • shift+dot to move right indenting
  • shift+, to move left indenting
  • to indent whole file gg (to move to top) , =, shift+g
